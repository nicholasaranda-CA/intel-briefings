How to Use These Briefings

Analytical Guidance for Students

  • Read for argument and assessment, not just facts — what does the analyst conclude?
  • Track which sources are cited and what perspective each brings
  • Note where sources agree or conflict — that tension is analytically valuable
  • Discussion questions at the end of each brief require original analysis, not recall
  • Print to PDF: File → Print → Save as PDF, Paper: Letter, Margins: Minimum
  • Briefings are updated as situations develop — check back for revised versions
